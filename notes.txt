Kataloog:
 * API url: https://eits.ria.ee/api/2/catalog/2023 - tagastab kataloogi kust saab moodulite struktuur.
 * Swagger skeema: /docs/eits-api-outputs/catalog_schema.png
	- Shema says that response contains module methods as well, but out put is only till group/module level.  So methods information should be retrieved from somewhere else.
 * Näidis 2023 veersiooni päringu väljund: /docs/eits-api-outputs/catalog-2023.json

Artikkel:
 * API utl: tagastab kogu kontenti sisu puu struktuuris
 * Näidis 2023 versiooni päringu tulemus: /docs/eits-api-outputs/article-2023.json

Esialgne lähenemine. Leida naturaalvõtit ja selle järgi sisustada meetheid: 
	1. Teha kataloogi päringu, et kätte saada moodulite nimekiri. Sealt saab väljalugeda moodulite gruppide koodid.
		* Omab kindla struktuuri
		* Saab väjalugeda koodid
	2. Teha artikli päringu ja siis sealt välja lugeda moodulite meetmed ja nende sisu
	3. Agregeeruda andmestiku ühise tabli.
	4. Kuvada tabeli siusu tabelis.
